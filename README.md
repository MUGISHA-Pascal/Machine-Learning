# Machine Learning Repository

A collection of machine learning projects, tutorials, and experiments.

## Repository Structure

```
.
├── notebooks/              # Jupyter notebooks organized by purpose
│   ├── tutorials/         # Learning materials and algorithm implementations
│   │   ├── algorithms/    # Linear regression, MNIST, and other algorithms
│   │   ├── backpropagation/
│   │   ├── computer-vision/
│   │   └── convolutional-neural-network/
│   ├── examples/          # Dataset examples and demonstrations
│   │   ├── breastCancer.ipynb
│   │   ├── CaliforniaHousing.ipynb
│   │   └── SVM(irisdataset).ipynb
│   ├── experiments/       # Test notebooks and custom implementations
│   └── visualization/     # Data visualization notebooks and resources
└── projects/              # Production-ready ML projects
    ├── breast-cancer-project/
    ├── california-housing-project/
    ├── diabetes-project/
    ├── irisFeature-project/
    ├── music-genre-generation-project/
    ├── student-grade-project/
    ├── videoGame-project/
    └── wine-project/
```

## Getting Started

1. Navigate to `notebooks/tutorials/` for learning materials
2. Check `notebooks/examples/` for dataset-specific implementations
3. Explore `projects/` for complete ML applications

## Projects

Each project folder contains:
- Training scripts (`train.py`)
- Flask API (`app.py`)
- Trained models (`model/`)
- Demo applications (`demo/`, `nodeApp/`)
